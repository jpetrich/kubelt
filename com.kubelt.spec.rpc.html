path

RPC calls as defined in an OpenRPC schema are strings. We prefer touse a vector of keywords, which we get by breaking apart the callname at each underscore and converting the resulting strings tokeywords. Converting to structured data makes them nicer tomanipulate when we want to do things like adding a prefix.
